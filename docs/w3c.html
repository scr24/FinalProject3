W3C And HTML And CSS
brief intro goes here

Title goes here

W3C is the World Wide Web Consortium. This organization exists to create standards for the web. They make decisions such as which protocols are useful and should be standardized and which conventions to stick to using. HTML is Hyper Text Markup Language. This is the language used to construct a webpage. It can structure the titles, paragraphs, content, etc within a web page. CSS stands for Cascading Style Sheets. CSS augments HTML in that it's used to design HTML pages. This allows for simple design changes across the board in regards to fonts, sizes, colors, etc.
